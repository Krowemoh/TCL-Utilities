# npm
Package Manager

## Installation Instructions

Download NPM

```
curl https://raw.githubusercontent.com/Krowemoh/npm/main/NPM -o /home/SYSPROG/BP/NPM
```

Compile and catalog NPM

```
BASIC BP NPM
CATALOG BP NPM
```

## Install Packages

```
NPM INSTALL BP NSH
NPM INSTALL BP EVA
```

## List Packages

```
NPM LIST
```
